# sailplane-data
The purpose of this repository is to present useful glider/sailplane data in searchable and maintainable ways.  Newer pilots may find this information useful while learning how to prepare for and plan flights. Seasoned pilots may find this information to be equally useful for upskilling and performance tuning purposes.

## Disclaimer
All Data and other information provided herein are for informational purposes only, and may contain errors as introduced by its authors. Users should reference only official aircraft publications for all flight planning purposes as required by the various governing bodies. Use this data at your own risk.
